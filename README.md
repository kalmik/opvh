# opvh
Opensips Pseudo Variables Helper.

```
 $ opvh --help
usage: opvh [-h] [-V DOC_VERSION] [-f] pv

positional arguments:
  pv

optional arguments:
  -h, --help            show this help message and exit
  -V DOC_VERSION, --doc-version DOC_VERSION
                        Specify version
  -f, --force-fetch     Force documentation fetch.
```
